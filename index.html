Subs.ro Kodi Repository

Addon Kodi pentru subtitrari de pe subs.ro — cea mai mare baza de date de subtitrari romanesti.

Kodi Source URL (adauga in File Manager):
https://triffalin.github.io/repository.subsro/

Fisiere disponibile

repository.subsro-1.0.0.zip service.subtitles.subsro-1.0.0.zip addons.xml addons.xml.md5

Instalare

  1. Settings → File Manager → Add Source
    URL: https://triffalin.github.io/repository.subsro/
  2. Addons → Install from zip file → selecteaza sursa → repository.subsro-1.0.0.zip
  3. Addons → Install from repository → Subs.ro Repository → Subtitle providers → Subs.ro
  4. Configureaza API key de pe subs.ro (Profil → API)
